## Overview

This site. A small digital garden where projects, thoughts, and reading all live together, every card driven by a Markdown file you can edit in seconds.

## What I built

A zero-build content system on top of a lightweight React runtime: drop a `.md` into a folder, push, and it appears. *No tokens in the browser*, fully static, hosted on GitHub Pages.

## Outcome

The goal was a site I'd actually keep updating. Lowering the cost of adding a thing to *"write a paragraph"* was the whole point.
